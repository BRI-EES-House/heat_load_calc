A.1 規模係数

　規模係数 \(r_{size}\) は式(1)による。

\[r_{size} = \frac{A_{f,total}}{A_{s,f,total}} \tag{1}\]

　ここで、
　\(r_{size}\)：規模係数
　\(A_{f,total}\)：床面積の合計, m2
　\(A_{s,f,total}\)：簡易モデル住戸の床面積の合計, m2
である。ここで、簡易モデル住戸の床面積の合計\(A_{s,f,total}\)は、90.0(m2)とする。

A.2 主開口方位

A.4.2 壁面からの日除けの張り出し寸法

　壁面から日除けの張り出し寸法\(z\)は式(3)による。
（この式は、もう少しきちんと考察する必要がある。（特に（3a）））

1地域～7地域

\[\begin{equation} z = \frac{z_h + z_c }{ 2 } \tag{3a} \end{equation}\]

8地域

\[\begin{equation} z = z_c \tag{3b} \end{equation}\]

　ここで、
　\(z\)：壁面から日除けの張り出し寸法, m
　\(z_h\)：暖房期の壁面から日除けの張り出し寸法, m
　\(z_c\)：冷房期の壁面から日除けの張り出し寸法, m
である。

　暖房期及び冷房期の壁面から日除けの張り出し寸法\(z_h\)・\(z_c\)は、暖房期及び冷房期の取得日射熱補正係数を指定する場合は次の(1)により、指定しない場合は(2)による。

(1) 取得日射熱補正係数を指定する場合

\(\frac{f_{s,h}}{0.01} - a_h \le 0\)の場合

\[\begin{equation} z_h = 5.0 \tag{4a} \end{equation}\]

\(\frac{f_{s,h}}{0.01} - a_h \gt 0\)の場合

\[\begin{equation} z_h= \left\{ \begin{array}{ll} b_h ( c_h y_1 + y_2 ) / ( \frac{f_{s,h} }{0.01} - a_h ) & ( f_{s,h} \le 0.72 ) \\ b_h ( c_h y_1 + y_2 ) / ( \frac{0.72}{0.01} - a_h ) & ( f_{s,h} \gt 0.72 ) \end{array} \right. \tag{4b} \end{equation}\]

　ただし、\(z_h\)が5.0を超える場合は\(z_h=5.0\)とする。

\(\frac{f_{s,c}}{0.01} - a_c \le 0\)の場合

\[\begin{equation} z_c = 5.0 \tag{4c} \end{equation}\]

\(\frac{f_{s,c}}{0.01} - a_c \gt 0\)の場合

\[\begin{equation} z_c = \left\{ \begin{array}{ll} b_c ( c_c y_1 + y_2 ) / ( \frac{f_{s,c} }{0.01} - a_c ) & ( f_{s,c} \le 0.93 ) \\ b_c ( c_c y_1 + y_2 ) / ( \frac{0.93}{0.01} - a_c ) & ( f_{s,c} \gt 0.93 ) \end{array} \right. \tag{4d} \end{equation}\]

　ここで、
　\(z_h\)：暖房期の壁面から日除けの張り出し寸法, m
　\(z_c\)：冷房期の壁面から日除けの張り出し寸法, m
　\(y_1\)：日除け下端から窓上端までの垂直方向の距離(\(=0.0\)), m
　\(y_2\)：窓の開口高さ寸法(\(=1.1\)), m
　\(f_{s,h}\)：簡易モデル住戸における暖房期の取得日射熱補正係数
　\(f_{s,c}\)：簡易モデル住戸における冷房期の取得日射熱補正係数
である。

　式(4)における係数\(a_h\)・\(b_h\)・\(c_h\)及び\(a_c\)・\(b_c\)・\(c_c\)は、暖房期・冷房期の別・地域の区分・方位に応じて表3(a)及び表3(b)に与えられる。

　\(A_{evlp,i}\)：外皮の部位（一般部位・開口部）\(i\)の面積, m2
　\(l_{evlp,mr,i}\)：主たる居室に接する外皮の部位（熱橋）\(i\)の長さ, m
　\(l_{evlp,or,i}\)：その他の居室に接する外皮の部位（熱橋）\(i\)の長さ, m
　\(l_{evlp,nr,i}\)：非居室に接する外皮の部位（熱橋）\(i\)の長さ, m
　\(l_{evlp,i}\)：外皮の部位（熱橋）\(i\)の長さ, m

X.5.1 概要

　太陽高度と太陽方位角を計算する。

X.5.2 均時差と赤緯

　均時差と赤緯は、式(1)から式(7)による。

\[\begin{equation} n = Y - 1968 \tag{1} \end{equation}\]
\[\begin{equation} d_0 = 3.71 + 0.2596 n - INT \left[ \frac{n+3}{4} \right] \tag{2} \end{equation}\]
\[\begin{equation} M = 2 \pi \cdot \frac{(D-d_0)}{D_{ay}} \tag{3} \end{equation}\]
\[\begin{equation} \epsilon = \frac{2 \pi}{360} \cdot \left( 12.3901 + 0.0172 \cdot \left( n + \frac{M}{2 \pi} \right) \right) \tag{4} \end{equation}\]
\[\begin{equation} v = M + \frac{2 \pi}{360} \cdot \left( 1.914 \cdot \sin{M} + 0.02 \cdot \sin{(2M)} \right) \tag{5} \end{equation}\]
\[\begin{equation} E_t = (M - v) - \tan^{-1} \left( \frac{ 0.043 \cdot \sin {2( v + \epsilon )} }{ 1 - 0.043 \cdot \cos {2(v + \epsilon)} } \right) \tag{6} \end{equation}\]
\[\begin{equation} \sin{\delta} = \cos{(v+\epsilon)} \cdot \sin{\delta_0} \tag{7a} \end{equation}\]
\[\begin{equation} \cos{\delta} = \sqrt{1 - \sin^2{\delta}} \tag{7b} \end{equation}\]

\(INT\) は小数点以下切り捨てた整数を表す。

　ここで、
　\(n\)：1968年との年差
　\(Y\)：西暦年(=1989)
　\(d_0\)：平均軌道上の近日点通過日（暦表時による1968年1月1日正午基準の日差）, d
　\(M\)：平均近点離角, rad
　\(D\)：年通算日（1/1を \(D=1\) とする), d
　\(D_{ay}\)：近点年（近日点基準の公転周期日数）(=365.2596), d
　\(\epsilon\)：近日点と冬至点の角度, rad
　\(v\)：真近点離角, rad
　\(E_t\)：均時差, rad
　\(\delta\)：赤緯, rad
　\(\delta_0\)：北半球の冬至の日赤緯(=-23.4393[deg]), deg
である。

X.5.3 時角

\[\begin{equation} t = 2 \pi \cdot \frac{T_m - 12}{24} + (L - L_0) + E_t \tag{8} \end{equation}\]

ここで、
　\(t\)：時角, rad
　\(T_m\)：標準時, h
　\(L\)：経度, rad
　\(L_0\)：標準時の地点の経度（標準子午線）, rad
　\(E_t\)：均時差, rad
である。

X.5.4 太陽位置

　太陽高度 \(h_s\) は式(9)、太陽方位角 \(A_s\) は式(10)による。

\[\begin{equation} \sin{h_s} = \max{ \left( 0, \sin{\phi} \sin{\delta}⁡ + \cos{\phi} \cos{\delta}⁡ \cos{t} \right) ⁡} \tag{9a} \end{equation}\]
\[\begin{equation} \cos{h_s} = \sqrt{1 - \sin^2{h_s}} \tag{9b} \end{equation}\]
\[\sin A_s = \left\{ \begin{array}{ll} \frac{ \cos \delta \sin t }{ \cos h_s } & ( h_s \gt 0 ) \\ 0 & ( h_s = 0 ) \end{array} \right. \tag{10a}\]
\[\cos A_s = \left\{ \begin{array}{ll} \frac { \sin h_s \sin \phi - \sin \delta }{ \cos h_s \cos \phi } & ( h_s \gt 0 ) \\ 0 & ( h_s = 0 ) \end{array} \right. \tag{10b}\]

ここで、
　\(h_s\)：太陽高度, rad
　\(A_s\)：太陽方位角, rad
　\(\phi\)：緯度, rad
　\(\delta\)：赤緯, rad
　\(t\)：時角, rad
である。 === X.6 外表面における入射角の方向余弦

\[\begin{align} \cos⁡ \phi &= \sin⁡{h_s} \cos{W_{\beta}}⁡ + \cos{h_s} \sin{W_{\beta}} \cos{(A_s - W_{\alpha})} \nonumber \\ &= \sin⁡{h_s} \cos{W_{\beta}}⁡ + \cos{h_s} \sin{W_{\beta}} \cos{A_s} \cos{W_{\alpha}} + \cos{h_s} \sin{W_{\beta}} \sin{A_s} \sin{W_{\alpha}} \tag{1} \end{align}\]

ただし、\(\cos{\phi} \lt 0\) の場合、\(\cos{\phi} = 0\) とする。

　ここで、
　\(\phi\)：外表面における入射角
　\(h_s\)：太陽高度, rad
　\(A_s\)：太陽方位角, rad
　\(W_{\alpha}\)：外表面の方位角, rad
　\(W_{\beta}\)：外表面の傾斜角, rad
である。 === X.10 窓の入射角特性

\[\begin{align} \cos⁡ \theta &= \sin⁡{h_s} \cos{W_{\beta}}⁡ + \cos{h_s} \sin{W_{\beta}} \cos{(A_s - W_{\alpha})} \nonumber \\ &= \sin⁡{h_s} \cos{W_{\beta}}⁡ + \cos{h_s} \sin{W_{\beta}} \cos{A_s} \cos{W_{\alpha}} + \cos{h_s} \sin{W_{\beta}} \sin{A_s} \sin{W_{\alpha}} \tag{1} \end{align}\]

ただし、\(\cos{\theta} \lt 0\) の場合、\(\cos{\theta} = 0\) とする。

　ここで、
　\(\theta\)：外表面における入射角
　\(h_s\)：太陽高度, rad
　\(A_s\)：太陽方位角, rad
　\(W_{\alpha}\)：外表面の方位角, rad
　\(W_{\beta}\)：外表面の傾斜角, rad
である。
