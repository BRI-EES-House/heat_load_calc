日よけの影面積の比率
************************************************************************************************************************

========================================================================================================================
I. 評価法
========================================================================================================================

1. 適用範囲

開口部の周辺の日よけが、開口部に到達する直達日射、天空日射を低減する効果を計算する。
計算する対象となる日よけは、開口部上部に横方向に無限にのびるひさし、および、開口部上部の横方向が有限のひさし(オーバーハング)と開口部
側方のひさし(サイドフィン)の組み合わせのいずれかである。

2. 開口部上部に横方向に無限にのびるひさしがある場合

2.1 直達日射の低減率の計算式

横方向に無限にのびるひさしがある場合には、ステップ |n| における開口部 |j| の直達日射に対する日よけの影面積比率 :math:`f_{ss,d,j,n}` は、式(1)で表される。

.. math::
    :nowrap:

    \begin{align*}
        f_{ss,d,j,n} = min\{ max\{ D_{h,j,n} / y_w, 0 \}, 1\} \tag{1}
    \end{align*}

ここで、

:math:`f_{ss,d,j,n}`
    | ステップ |n| における開口部 |j| の直達日射に対する日よけの影面積比率, -
:math:`y_w`
    | 開口部 |j| の高さ, m
:math:`D_{h,j,n}`
    | ステップ |n| に開口部 |j| に影がかかる長さ, m

である。なお、直達日射に対する日よけの影面積比率 :math:`f_{ss,d,j,n}` が :math:`0` となるのは窓上端に日陰がかからない場合であり、 :math:`1` となるのは窓下まで影になる場合である。

ステップ |n| に開口部 |j| に影がかかる長さ(窓上端から下方への長さ) :math:`D_{h,j,n}` は、式(2)で表される。

.. math::
    :nowrap:

    \begin{align*}
        D_{h,j,n} = z \tan \phi_{j,n} - y_1 \tag{2}
    \end{align*}

ここで、

:math:`y_1`
    | 開口部 |j| の上端からひさしまでの長さ, m
:math:`z`
    | ひさしの出幅, m
:math:`\phi_{j,n}`
    | ステップ |n| における開口部 |j| に対する太陽のプロファイル角, rad

である。
    
ステップ |n| における開口部 |j| に対する太陽のプロファイル角 :math:`\phi_{j,n}` の正接は、式(3)で表される。

.. math::
    :nowrap:

    \begin{align*}
       \tan \phi_{j,n} = \begin{cases}
           \tan h_{s,n} / \cos ( a_{s,n} - \alpha_{w,j} ) & ( a_{s,n} - \alpha_{w,j} \neq 0 ) \\
           0 & ( a_{s,n} - \alpha_{w,j} = 0 )
    	\end{cases}
    	\tag{3}
    \end{align*}

ここで、

:math:`h_{s,n}`
    | ステップ |n| における太陽高度, rad
:math:`a_{s,n}`
    | ステップ |n| における太陽方位角, rad
:math:`\alpha_{w,j}`
    | 開口部 |j| の方位角, rad

である。ただし、ステップ |n| において日が沈んでいる場合は、 :math:`h_{s,n}=0` 及び :math:`a_{s,n}=0` とする。


2.2 天空日射の低減率の計算式

無限にのびるひさしがある場合には、開口部 |j| の天空放射に対する日よけによる低減率 :math:`f_{ss,s,j}` は、式(2)で表される。

.. math::
    :nowrap:

    \begin{align*}
        f_{ss,s,j} = \dfrac{ \{ (y_1 + y_w) + \sqrt{ {y_1}^2 + z^2 } \} - \{ y_1 + \sqrt{ (y_1 + y_w)^2 + z^2 } \} } {2 y_w} \tag{2}
    \end{align*}

ここで、

:math:`f_{ss,s,j}`
    | 開口部 |j| の天空放射に対する日よけによる低減率, -
:math:`y_1`
    | 窓の上端から庇までの長さ, m
:math:`y_w`
    | 窓の高さ, m
:math:`z`
    | ひさしの出幅, m
    
である。天空放射に対する日よけによる低減率 :math:`f_{ss,s,j}` は、ひさしが無い場合で :math:`0` を、ひさしが無限に出ている仮想の場合で :math:`0.5` の値をとる。
