隙間風

========================================================================================================================
I. 評価法
========================================================================================================================

2階建てまでに適用できる住宅用の近似式を使用し、すきま風量を計算する。

室 |i| のすきま風量 :math:`V_{infiltration,i}` は式(1)で表される。

.. math::
    :nowrap:

    \begin{align*}
    	  V_{infiltration,i} = n_{infiltration} V_{room,i} /3600 \tag{1}
    \end{align*}

ここで、

:math:`V_{infiltration,i}`
    | 室 |i| のすきま風量, |m3| /s
:math:`n_{infiltration}`
    | すきま風による住宅全体の換気回数, 回/h
:math:`V_{room,i}`
    | 室 |i| の容積, |m3| 
    
である。

すきま風による住宅全体の換気回数 :math:`n_{infiltration}` は式(2)で表される。

.. math::
    :nowrap:

    \begin{align*}
    	  n_{infiltration} = \max \{ a \cdot C \sqrt{\Delta \theta} - b, 0 \} \tag{2}
    \end{align*}

ここで、

:math:`a`
    | 近似式の係数, 回/(h cm\ :sup:`2`\/ |m2| K\ :sup:`0.5`\)
:math:`b`
    | 近似式の係数, 回/h
:math:`C`
    | 相当隙間面積, cm\ :sup:`2`\/ |m2| 
    
である。

近似式の係数 :math:`a` は階数で決定される係数であり、1階建ての時に :math:`0.022` を、2階建ての時に :math:`0.020` をとる。

近似式の係数 :math:`b` は階数と換気方式の組み合わせで決定される係数であり、室内の圧力が高くなる二種換気では、1階建てで :math:`b = 0.26` に、2階建てで :math:`b = 0.14` となる。室内の圧力が低くなる三種換気では、1階建てで :math:`b = 0.28` を、2階建てで :math:`b = 0.13` となる。内外差圧がバランスしている一種換気では、階数によらず :math:`b = 0` となる。
