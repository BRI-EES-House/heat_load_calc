傾斜面の方位角・傾斜角
************************************************************************************************************************

========================================================================================================================
I. 評価法
========================================================================================================================

1. 適用範囲

傾斜面の方位角・傾斜角を方向名称から求める。

2. 計算式

室 |i| の境界 |j| の方位角 :math:`\alpha_{w,i,j}` は、境界 |j| の向きに応じて、以下のように定義される。

* 境界 |j| が南を向いている壁の場合： :math:`\alpha_{w,i,j} = 0`
* 境界 |j| が南西を向いている壁の場合： :math:`\alpha_{w,i,j} = \pi/4`
* 境界 |j| が西を向いている壁の場合： :math:`\alpha_{w,i,j} = \pi/2`
* 境界 |j| が北西を向いている壁の場合： :math:`\alpha_{w,i,j} = 3 \pi/4`
* 境界 |j| が北を向いている壁の場合： :math:`\alpha_{w,i,j} = \pi`
* 境界 |j| が北東を向いている壁の場合： :math:`\alpha_{w,i,j} = - 3 \pi/4`
* 境界 |j| が東を向いている壁の場合： :math:`\alpha_{w,i,j} = - \pi/2`
* 境界 |j| が南東を向いている壁の場合： :math:`\alpha_{w,i,j} = - \pi/4`
* 境界 |j| が上を向いている屋根・天井の場合： :math:`\alpha_{w,i,j} = 0`
* 境界 |j| が下を向いている床の場合： :math:`\alpha_{w,i,j} = 0`

ここで、

:math:`\alpha_{w,i,j}`
    | 室 |i| の境界 |j| の方位角`, rad 
    
である。

室 |i| の境界 |j| の傾斜角 :math:`\beta_{w,i,j}` は、境界 |j| の向きに応じて、以下のように定義される。

* 境界 |j| が南を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が南西を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が西を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が北西を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が北を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が北東を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が東を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が南東を向いている壁の場合： :math:`\beta_{w,i,j} = \pi/2`
* 境界 |j| が上を向いている屋根・天井の場合： :math:`\beta_{w,i,j} = 0`
* 境界 |j| が下を向いている床の場合： :math:`\beta_{w,i,j} = \pi`

ここで、

:math:`\beta_{w,i,j}`
    | 室 |i| の境界 |j| の傾斜角`, rad 
    
である。
